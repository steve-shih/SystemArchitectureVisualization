資料處理在做什麼？

模型不是直接「理解資料」，而是吃進數值張量（Tensor）。資料處理就是把雜亂、缺漏、尺度不一、類別文字、圖片、文字、時間序列，轉成模型容易學習的格式。

核心目標：降低噪音、避免資料洩漏、穩定訓練、提升泛化能力。

一、資料清理 Data Cleaning

缺失值補齊 Missing Value Imputation

清理

常見方法：平均數、中位數、眾數、KNN Imputer、模型預測補值。實務上中位數對極端值較穩。

from sklearn.impute import SimpleImputer
imputer = SimpleImputer(strategy="median")
X_fixed = imputer.fit_transform(X)

離群值偵測 Outlier Detection

清理

常見方法：Z-Score、IQR、Isolation Forest、Local Outlier Factor。適合先找異常資料，再決定刪除、截尾或保留。

q1, q3 = df["price"].quantile([.25, .75])
iqr = q3 - q1
mask = df["price"].between(q1 - 1.5*iqr, q3 + 1.5*iqr)

二、類別編碼 Categorical Encoding

One-Hot Encoding

編碼

將類別拆成多個 0/1 欄位。適合低基數類別，例如性別、城市、商品類型。

pd.get_dummies(df, columns=["city"])

Embedding 類別向量化

深度學習

把高基數類別轉成可學習向量，例如 user_id、product_id、word_id。常用於推薦系統與 NLP。

embedding = torch.nn.Embedding(num_embeddings=10000, embedding_dim=32)
vector = embedding(category_id)

三、特徵縮放 Feature Scaling

Standardization 標準化

縮放

轉成平均值 0、標準差 1。深度學習、線性模型、SVM、KNN 常需要。

from sklearn.preprocessing import StandardScaler
X_scaled = StandardScaler().fit_transform(X)

Min-Max Scaling

縮放

將資料壓到 0～1。適合已知上下界或影像像素正規化。

X_scaled = (X - X.min()) / (X.max() - X.min())

四、資料切分 Dataset Split

Train / Validation / Test Split

驗證

訓練集學習、驗證集調參、測試集做最終評估。最重要的是避免資料洩漏。

train, test = train_test_split(df, test_size=0.2, random_state=42)

K-Fold Cross Validation

驗證

把資料切成 K 份，輪流當驗證集。資料量少時比單次切分穩定。

from sklearn.model_selection import KFold
for train_idx, valid_idx in KFold(n_splits=5).split(X):
    pass

五、特徵工程與降維 Feature Engineering / Dimensionality Reduction

PCA 主成分分析

降維

找出資料變異最大的方向，把高維資料投影到低維空間。常用於壓縮、去噪、視覺化。

from sklearn.decomposition import PCA
X_2d = PCA(n_components=2).fit_transform(X)

Polynomial Features 多項式特徵

特徵工程

把 x 轉成 x、x²、x³，讓線性模型能擬合非線性關係。

from sklearn.preprocessing import PolynomialFeatures
X_poly = PolynomialFeatures(degree=2).fit_transform(X)

六、不平衡資料 Imbalanced Data

SMOTE 合成少數類樣本

分類資料

在少數類樣本之間插值產生新樣本。適合分類問題，但不能在切分前做，否則會資料洩漏。

from imblearn.over_sampling import SMOTE
X_resampled, y_resampled = SMOTE().fit_resample(X_train, y_train)

Class Weight 類別權重

訓練策略

不改資料量，而是提高少數類錯誤成本。常用於 Logistic Regression、XGBoost、PyTorch loss。

loss_fn = torch.nn.CrossEntropyLoss(weight=class_weights)

七、文字資料處理 NLP Preprocessing

Tokenization 斷詞 / 分詞

NLP

把句子拆成 token。傳統 NLP 可能用單字，Transformer 常用 BPE / WordPiece / SentencePiece。

tokens = tokenizer("我喜歡機器學習", return_tensors="pt")

TF-IDF 文字向量化

NLP

常見於傳統 ML。常出現但不是每篇都有的詞，權重會比較高。

from sklearn.feature_extraction.text import TfidfVectorizer
X = TfidfVectorizer().fit_transform(texts)

八、影像資料處理 Image Preprocessing / Augmentation

影像增強 Flip / Crop / Rotate

CV

在不改變標籤的前提下製造變化，讓模型不要死背位置、角度、亮度。

transforms.Compose([
    transforms.RandomHorizontalFlip(),
    transforms.RandomRotation(15),
    transforms.ToTensor()
])

Image Normalization 像素正規化

CV

把 0～255 像素轉成 0～1 或依通道做 mean/std normalization，讓 CNN 訓練更穩。

image = image / 255.0
image = (image - mean) / std

九、時間序列處理 Time Series Preprocessing

Sliding Window 滑動視窗

時間序列

用前 N 個時間點預測下一個值。常用於股價、流量、感測器、伺服器監控。

def make_window(values, window_size):
    X, y = [], []
    for i in range(len(values) - window_size):
        X.append(values[i:i+window_size])
        y.append(values[i+window_size])
    return X, y

Rolling Statistics 移動平均 / 統計特徵

時間序列

把最近 N 天平均、最大、最小、標準差當成新特徵，常用於表格型時間序列。

df["rolling_mean_7"] = df["value"].rolling(7).mean()

完整 Pipeline 範例

正式專案要注意：所有 fit 都只能在 train 做，再 transform validation/test，否則會資料洩漏（Data Leakage）。

from sklearn.compose import ColumnTransformer
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import OneHotEncoder, StandardScaler
from sklearn.impute import SimpleImputer
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier

numeric_features = ["age", "income", "visit_count"]
categorical_features = ["city", "device"]

numeric_pipeline = Pipeline([
    ("imputer", SimpleImputer(strategy="median")),
    ("scaler", StandardScaler())
])

categorical_pipeline = Pipeline([
    ("imputer", SimpleImputer(strategy="most_frequent")),
    ("onehot", OneHotEncoder(handle_unknown="ignore"))
])

preprocess = ColumnTransformer([
    ("numeric", numeric_pipeline, numeric_features),
    ("categorical", categorical_pipeline, categorical_features)
])

model = Pipeline([
    ("preprocess", preprocess),
    ("classifier", RandomForestClassifier(random_state=42))
])

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=.2, stratify=y)
model.fit(X_train, y_train)
score = model.score(X_test, y_test)
